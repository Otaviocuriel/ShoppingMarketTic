# 🛒 ShoppingMarketTic

Aplicação de e-commerce desenvolvida com React, TypeScript e Tailwind CSS, com funcionalidades de carrinho de compras e filtro de produtos por preço.

-----

## ✨ Funcionalidades

- 🛍️ Listagem de produtos
- 🔽 Filtro de produtos por menor e maior valor
- 🛒 Carrinho de compras (adicionar, remover e visualizar itens)

-----

## 🚀 Tecnologias

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Vite](https://vitejs.dev/)
- [ESLint](https://eslint.org/) + [Prettier](https://prettier.io/)

-----

## 🖥️ Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/Otaviocuriel/ShoppingMarketTic.git

# Entre na pasta do projeto
cd ShoppingMarketTic

# Instale as dependências
yarn server

# Inicie o servidor de desenvolvimento
yarn dev
```

Acesse em: `http://localhost:5173`
